# @solidjs-email/body

[![npm version](https://img.shields.io/npm/v/@solidjs-email/body.svg)](https://www.npmjs.com/package/@solidjs-email/body)

Body component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/body
```

## Usage

```tsx
import { ubody } from '@solidjs-email/body';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
