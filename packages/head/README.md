# @monkedevlife/solidjs-email-head

[![npm version](https://img.shields.io/npm/v/@monkedevlife/solidjs-email-head.svg)](https://www.npmjs.com/package/@monkedevlife/solidjs-email-head)

Head component for SolidJS email templates

## Installation

```bash
npm install @monkedevlife/solidjs-email-head
```

## Usage

```tsx
import { uhead } from '@monkedevlife/solidjs-email-head';
```

See the [main documentation](https://github.com/monkedevlife/solidjs-email) for full usage examples.
