# @solidjs-email/head

[![npm version](https://img.shields.io/npm/v/@solidjs-email/head.svg)](https://www.npmjs.com/package/@solidjs-email/head)

Head component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/head
```

## Usage

```tsx
import { uhead } from '@solidjs-email/head';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
