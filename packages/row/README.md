# @solidjs-email/row

[![npm version](https://img.shields.io/npm/v/@solidjs-email/row.svg)](https://www.npmjs.com/package/@solidjs-email/row)

Row component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/row
```

## Usage

```tsx
import { urow } from '@solidjs-email/row';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
