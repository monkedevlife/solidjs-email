# @monkedevlife/solidjs-email-column

[![npm version](https://img.shields.io/npm/v/@monkedevlife/solidjs-email-column.svg)](https://www.npmjs.com/package/@monkedevlife/solidjs-email-column)

Column component for SolidJS email templates

## Installation

```bash
npm install @monkedevlife/solidjs-email-column
```

## Usage

```tsx
import { ucolumn } from '@monkedevlife/solidjs-email-column';
```

See the [main documentation](https://github.com/monkedevlife/solidjs-email) for full usage examples.
