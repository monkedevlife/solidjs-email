# @solidjs-email/column

[![npm version](https://img.shields.io/npm/v/@solidjs-email/column.svg)](https://www.npmjs.com/package/@solidjs-email/column)

Column component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/column
```

## Usage

```tsx
import { ucolumn } from '@solidjs-email/column';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
