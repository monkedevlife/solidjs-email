# @solidjs-email/hr

[![npm version](https://img.shields.io/npm/v/@solidjs-email/hr.svg)](https://www.npmjs.com/package/@solidjs-email/hr)

Horizontal rule component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/hr
```

## Usage

```tsx
import { uhr } from '@solidjs-email/hr';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
