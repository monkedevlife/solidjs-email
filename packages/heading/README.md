# @solidjs-email/heading

[![npm version](https://img.shields.io/npm/v/@solidjs-email/heading.svg)](https://www.npmjs.com/package/@solidjs-email/heading)

Heading component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/heading
```

## Usage

```tsx
import { uheading } from '@solidjs-email/heading';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
