# @monkedevlife/solidjs-email-heading

[![npm version](https://img.shields.io/npm/v/@monkedevlife/solidjs-email-heading.svg)](https://www.npmjs.com/package/@monkedevlife/solidjs-email-heading)

Heading component for SolidJS email templates

## Installation

```bash
npm install @monkedevlife/solidjs-email-heading
```

## Usage

```tsx
import { uheading } from '@monkedevlife/solidjs-email-heading';
```

See the [main documentation](https://github.com/monkedevlife/solidjs-email) for full usage examples.
