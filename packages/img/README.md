# @solidjs-email/img

[![npm version](https://img.shields.io/npm/v/@solidjs-email/img.svg)](https://www.npmjs.com/package/@solidjs-email/img)

Image component for SolidJS email templates

## Installation

```bash
npm install @solidjs-email/img
```

## Usage

```tsx
import { uimg } from '@solidjs-email/img';
```

See the [main documentation](https://github.com/solidjs-email/solidjs-email) for full usage examples.
