# @monkedevlife/solidjs-email-img

[![npm version](https://img.shields.io/npm/v/@monkedevlife/solidjs-email-img.svg)](https://www.npmjs.com/package/@monkedevlife/solidjs-email-img)

Image component for SolidJS email templates

## Installation

```bash
npm install @monkedevlife/solidjs-email-img
```

## Usage

```tsx
import { uimg } from '@monkedevlife/solidjs-email-img';
```

See the [main documentation](https://github.com/monkedevlife/solidjs-email) for full usage examples.
